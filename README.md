# 🎬 Movie Library — Личная кинотека

> *"Храни фильмы, которые изменили твою жизнь"*

---

## 📖 О проекте

**Movie Library** — это графическое приложение для ведения личной базы фильмов. 
Добавляй, удаляй, фильтруй и анализируй свою кинотеку. Все данные сохраняются в JSON-файл.

---

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| ➕ Добавление фильма | Название, жанр, год, рейтинг (0-10) |
| 🗑️ Удаление | Удали фильм, который стыдно признать, что смотрел |
| 🔍 Фильтрация | По жанру и году выпуска |
| 📊 Статистика | Общее количество, средний рейтинг, лучший/худший фильм |
| 💾 Автосохранение | Всё сохраняется в JSON и загружается при запуске |
| 🛡️ Валидация | Защита от дурака: год — число, рейтинг — от 0 до 10 |
